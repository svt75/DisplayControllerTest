# Display Controller Test Tool

Windows приложение для тестирования LED табло DisplayController.

## Скачать

Готовый EXE файл доступен в разделе [Releases](../../releases).

## Функции

- Отправка текста на экран 1 (576×384)
- Отправка текста на экран 2 (288×192)
- Настройка размера шрифта для каждой строки
- Очистка экранов
- Включение/выключение часов
- Лог всех операций

## Использование

1. Укажите IP адрес плеера
2. Порт по умолчанию: 9854
3. Token уже заполнен
4. Нажмите **Test** для проверки соединения
5. Введите текст в строки
6. Нажмите **SEND** для отправки

## Кнопки

| Кнопка | Действие |
|--------|----------|
| Test | Проверить соединение |
| SEND | Отправить на экран |
| Clear | Очистить экран |
| Fill | Заполнить тестовыми данными |
| SEND BOTH | Отправить на оба экрана |
| CLEAR ALL | Очистить всё |
| CLOCK ON | Включить часы |
| CLOCK OFF | Выключить часы |

## API Endpoints

```
GET  /status      - статус
POST /set1        - данные экрана 1
POST /set2        - данные экрана 2
POST /clear       - очистить все
POST /clear1      - очистить экран 1
POST /clear2      - очистить экран 2
POST /clock       - запустить часы
POST /clock/stop  - остановить часы
```

## Формат данных

```json
{
  "line1": {"text": "Текст", "fontSize": 24},
  "line2": {"text": "Текст 2", "fontSize": 24}
}
```

## Сборка

Требуется .NET 6.0 SDK.

```bash
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true
```
